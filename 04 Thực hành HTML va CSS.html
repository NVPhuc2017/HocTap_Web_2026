📝 ĐỀ KIỂM TRA 1: HTML CƠ BẢN (Bài 1-5)

Kiểm tra kiến thức về cấu trúc HTML, thẻ cơ bản, danh sách, liên kết, bảng và form

Câu 1: Trang web giới thiệu bản thân (15 điểm)

Yêu cầu: Tạo trang web giới thiệu bản thân với các nội dung:

  • Tiêu đề trang: "Giới thiệu về tôi"
  • Thẻ h1: Họ tên của bạn
  • Một đoạn văn giới thiệu về bản thân (sở thích, đam mê)
  • Danh sách không thứ tự các kỹ năng (tối thiểu 3 kỹ năng)
  • Một hình ảnh đại diện (có thể dùng ảnh placeholder)
  • Liên kết đến trang mạng xã hội (mở tab mới)
<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <title>Giới thiệu về tôi</title>
</head>
<body>
    <h1>Nguyễn Văn A</h1>
    <img src="avatar-placeholder.jpg" alt="Ảnh đại diện" width="150">
    <p>Xin chào! Tôi là Nguyễn Văn A, hiện đang học lập trình web. Tôi đam mê thiết kế giao diện và xây dựng các ứng dụng web.</p>
    <h2>Kỹ năng</h2>
    <ul>
        <li>HTML5</li>
        <li>CSS3</li>
        <li>JavaScript cơ bản</li>
    </ul>
    <p>Kết nối với tôi: <a href="https://facebook.com" target="_blank">Facebook</a></p>
</body>
</html>

Câu 2: Bảng thời khóa biểu (15 điểm)

Yêu cầu: Tạo bảng thời khóa biểu với:

  • Hiển thị 5 ngày trong tuần (Thứ 2 đến Thứ 6)

📝 ĐỀ KIỂM TRA 2: HTML NÂNG CAO (Bài 6-10)

Kiểm tra kiến thức về form nâng cao, iframe, multimedia và semantic HTML

Câu 1: Form đăng ký khóa học (15 điểm)

Yêu cầu: Tạo form đăng ký khóa học với các trường:

  • Họ tên (text, required)
  • Email (email, required)
  • Số điện thoại (tel)
  • Khóa học muốn đăng ký (select với 3 lựa chọn: HTML/CSS, JavaScript, ReactJS)
  • Phương thức thanh toán (radio: Chuyển khoản / Tiền mặt)
  • Ghi chú (textarea)
  • Nút gửi và nút reset
<form>
    <label>Họ tên: </label><input type="text" name="fullname" required><br>
    <label>Email: </label><input type="email" name="email" required><br>
    <label>SĐT: </label><input type="tel" name="phone"><br>
    <label>Khóa học: </label>
    <select name="course">
        <option>HTML/CSS</option>
        <option>JavaScript</option>
        <option>ReactJS</option>
    </select><br>
    <label>Thanh toán: </label>
    <input type="radio" name="payment" value="bank"> Chuyển khoản
    <input type="radio" name="payment" value="cash"> Tiền mặt<br>
    <label>Ghi chú: </label><textarea rows="3"></textarea><br>
    <button type="submit">Gửi đăng ký</button>
    <button type="reset">Nhập lại</button>
</form>

Câu 2: Trang video gallery (15 điểm)

Yêu cầu: Tạo trang gallery video với:

  • Tiêu đề trang: "Video Gallery"
  • Nhúng 2 video từ YouTube bằng iframe
  • Nhúng 1 video local bằng thẻ video (có controls)
  • Nhúng 1 file audio (có controls)
  • Sử dụng semantic HTML: header, main, section, footer
<!DOCTYPE html>
<html>
<head><title>Video Gallery</title></head>
<body>
    <header><h1>Video Gallery</h1></header>
    <main>
        <section>
            <h2>YouTube Videos</h2>
            <iframe width="560" height="315" src="https://www.youtube.com/embed/dQw4w9WgXcQ"></iframe>
            <iframe width="560" height="315" src="https://www.youtube.com/embed/jfKfPfyJRdk"></iframe>
        </section>
        <section>
            <h2>Video Local</h2>
            <video width="560" controls>
                <source src="sample.mp4" type="video/mp4">
            </video>
        </section>
        <section>
            <h2>Audio</h2>
            <audio controls>
                <source src="sample.mp3" type="audio/mpeg">
            </audio>
        </section>
    </main>
    <footer><p>© 2024 Video Gallery</p></footer>
</body>
</html>

📝 ĐỀ KIỂM TRA 3: CSS CƠ BẢN (Bài 11-15)

Kiểm tra kiến thức về selector, box model, flexbox cơ bản

Câu 1: Profile Card (15 điểm)

Yêu cầu: Thiết kế profile card với CSS:

  • Card có width 300px, border-radius 15px, box-shadow
  • Ảnh đại diện hình tròn (width 100px, border-radius 50%)
  • Tên người dùng font-size 24px, màu xanh đậm
  • Mô tả công việc màu xám, font italic
  • Nút "Theo dõi" có hiệu ứng hover (đổi màu)
  • Card được căn giữa trang
<style>
    .profile-card {
        width: 300px;
        margin: 50px auto;
        background: white;
        border-radius: 15px;
        box-shadow: 0 10px 30px rgba(0,0,0,0.1);
        text-align: center;
        padding: 20px;
    }
    .avatar {
        width: 100px;
        height: 100px;
        border-radius: 50%;
        object-fit: cover;
    }
    h2 { color: #2c3e50; margin: 10px 0; }
    .job { color: #7f8c8d; font-style: italic; }
    .follow-btn {
        background: #3498db;
        color: white;
        border: none;
        padding: 10px 20px;
        border-radius: 25px;
        cursor: pointer;
        transition: background 0.3s;
    }
    .follow-btn:hover { background: #2980b9; }
</style>

Câu 2: Flexbox Navigation (15 điểm)

Yêu cầu: Tạo thanh navigation với Flexbox:

  • Thanh navigation có background gradient
  • Các mục menu: Trang chủ, Giới thiệu, Sản phẩm, Dịch vụ, Liên hệ
  • Menu căn giữa theo chiều ngang, khoảng cách 25px
  • Hover: đổi màu nền và có hiệu ứng chuyển động
  • Responsive: màn hình < 768px menu chuyển thành dọc
<style>
    nav {
        background: linear-gradient(135deg, #667eea, #764ba2);
        padding: 1rem;
    }
    .nav-menu {
        display: flex;
        justify-content: center;
        gap: 25px;
        list-style: none;
    }
    .nav-menu a {
        color: white;
        text-decoration: none;
        padding: 10px 20px;
        transition: all 0.3s;
    }
    .nav-menu a:hover {
        background: rgba(255,255,255,0.2);
        border-radius: 5px;
        transform: translateY(-2px);
    }
    @media (max-width: 768px) {
        .nav-menu {
            flex-direction: column;
            text-align: center;
        }
    }
</style>

📝 ĐỀ KIỂM TRA 4: CSS NÂNG CAO (Bài 16-20)

Kiểm tra kiến thức về Grid, animation, responsive design

Câu 1: Product Grid (15 điểm)

Yêu cầu: Tạo grid sản phẩm với CSS Grid:

  • Desktop: 4 cột, tablet: 2 cột, mobile: 1 cột
  • Mỗi card sản phẩm có hình ảnh, tên sản phẩm, giá, nút mua
  • Hiệu ứng hover: card nổi lên, đổ bóng đậm hơn
  • Sử dụng grid-gap: 20px
<style>
    .products-grid {
        display: grid;
        grid-template-columns: repeat(4, 1fr);
        gap: 20px;
        padding: 20px;
    }
    .product-card {
        background: white;
        border-radius: 10px;
        overflow: hidden;
        box-shadow: 0 5px 15px rgba(0,0,0,0.1);
        transition: all 0.3s;
    }
    .product-card:hover {
        transform: translateY(-10px);
        box-shadow: 0 15px 30px rgba(0,0,0,0.2);
    }
    @media (max-width: 992px) {
        .products-grid { grid-template-columns: repeat(2, 1fr); }
    }
    @media (max-width: 576px) {
        .products-grid { grid-template-columns: 1fr; }
    }
</style>

Câu 2: Animation & Keyframes (15 điểm)

Yêu cầu: Tạo hiệu ứng animation:

  • Tạo hiệu ứng fadeIn cho phần tử khi load trang
  • Tạo hiệu ứng nhấp nháy cho nút "Đặt hàng"
  • Tạo hiệu ứng loader (vòng tròn quay) khi đang tải
  • Sử dụng @keyframes và animation
<style>
    @keyframes fadeIn {
        from { opacity: 0; transform: translateY(20px); }
        to { opacity: 1; transform: translateY(0); }
    }
    .container { animation: fadeIn 0.5s ease; }
    
    @keyframes blink {
        0%, 100% { opacity: 1; }
        50% { opacity: 0.5; background: #e74c3c; }
    }
    .btn-order { animation: blink 1s infinite; }
    
    @keyframes spin {
        from { transform: rotate(0deg); }
        to { transform: rotate(360deg); }
    }
    .loader {
        width: 40px;
        height: 40px;
        border: 4px solid #f3f3f3;
        border-top: 4px solid #3498db;
        border-radius: 50%;
        animation: spin 1s linear infinite;
    }
</style>
