🎨 CSS - Cascading Style Sheets

📖 LÝ THUYẾT

CSS là gì? Ngôn ngữ định dạng giao diện web, giúp trang web trở nên đẹp mắt và chuyên nghiệp.

📌 Vai trò của CSS:

  • ✓ Tách biệt nội dung (HTML) và hình thức (CSS)
  • ✓ Tạo bố cục, màu sắc, font chữ, hiệu ứng
  • ✓ Thiết kế responsive trên mọi thiết bị
  • ✓ Tối ưu trải nghiệm người dùng

Ví dụ: Trước và sau khi có CSS

❌ Không CSS

Text đơn giản

✅ Có CSS

Text đẹp, gradient, shadow

📝 BÀI TẬP THỰC HÀNH

Câu 1: CSS viết tắt của từ gì? Ý nghĩa của từng từ?

Cascading Style Sheets - "Cascading" nghĩa là xếp tầng, "Style Sheets" là bảng định dạng.

Câu 2: Tại sao nên tách riêng CSS khỏi HTML?

Giúp code dễ bảo trì, tái sử dụng, và phân chia công việc rõ ràng giữa front-end developer.

📝 Cú pháp CSS cơ bản

📖 LÝ THUYẾT
selector {
    property: value;
}
/* Ví dụ cụ thể */
h1 {
    color: blue;
    font-size: 24px;
    text-align: center;
}

📌 Các cách nhúng CSS:

Cách Cú pháp Ưu điểm
Inline <h1 style="color: red;"> Dễ test nhanh
Internal <style>...</style> Dùng cho 1 trang
External <link rel="stylesheet" href="style.css"> Tái sử dụng, chuyên nghiệp

📝 BÀI TẬP THỰC HÀNH

Câu 1: Có mấy cách để nhúng CSS vào HTML? Kể tên và nêu ưu điểm của từng cách.

3 cách: Inline (dễ test), Internal (dùng cho 1 trang), External (tái sử dụng, chuyên nghiệp).

Câu 2: Hãy viết cú pháp CSS để đặt màu xanh cho tất cả thẻ <p> và cỡ chữ 18px.

p { color: blue; font-size: 18px; }

Câu 3: External CSS được liên kết với HTML bằng thẻ gì? Viết cú pháp.

<link rel="stylesheet" href="style.css">

Câu 4: Trong cú pháp CSS, "selector" và "property" có vai trò gì?

Selector: chọn phần tử HTML. Property: thuộc tính muốn thay đổi.

Câu 5: Nếu có nhiều cách nhúng CSS cùng lúc, cách nào có độ ưu tiên cao nhất?

Inline CSS có độ ưu tiên cao nhất, tiếp đến Internal, cuối cùng là External.

Câu 1: Thuộc tính nào dùng để thay đổi màu chữ? Viết cú pháp.

color: #ff0000; hoặc color: red;

Câu 2: Làm thế nào để căn giữa văn bản theo chiều ngang?

text-align: center;

Câu 3: Thuộc tính nào tạo viền bo tròn cho phần tử? Cho ví dụ bo góc 20px.

border-radius: 20px;

Câu 4: Làm sao để tạo hiệu ứng đổ bóng cho khối với độ mờ 0.3?

box-shadow: 0 5px 15px rgba(0,0,0,0.3);

Câu 5: Đơn vị rem và em khác nhau như thế nào? Nên dùng đơn vị nào cho responsive?

rem dựa trên font-size của thẻ html, em dựa trên phần tử cha. Nên dùng rem cho responsive.

📦 Box Model - Mô hình hộp

📖 LÝ THUYẾT

Mọi phần tử HTML đều là một hộp với 4 thành phần: Margin, Border, Padding, Content

📌 MARGIN (Khoảng cách ngoài)
🔲 BORDER (Viền)
📦 PADDING (Khoảng cách trong)
📄 CONTENT
Nội dung chính
.box {
    width: 200px;
    padding: 20px;
    border: 2px solid black;
    margin: 10px;
    /* Tổng chiều rộng = 200 + 40 + 4 + 20 = 264px */
}

📝 BÀI TẬP THỰC HÀNH

Câu 1: Box Model trong CSS gồm những thành phần nào? Sắp xếp từ trong ra ngoài.

Content → Padding → Border → Margin

Câu 2: Nếu một phần tử có width: 150px, padding: 10px, border: 2px, margin: 5px, tổng chiều rộng thực tế là bao nhiêu?

150 + 20 + 4 + 10 = 184px

Câu 3: Thuộc tính nào dùng để tạo khoảng cách giữa các phần tử với nhau?

Margin

Câu 4: box-sizing: border-box; có tác dụng gì? Khi nào nên dùng?

Làm cho width và height bao gồm cả padding và border, nên dùng để dễ tính toán bố cục.

Câu 5: Làm thế nào để tạo khoảng cách 15px bên trái của phần tử?

margin-left: 15px; hoặc padding-left: 15px; (tùy mục đích)

🔲 Flexbox - Bố cục linh hoạt

📖 LÝ THUYẾT

Flexbox là mô hình bố cục 1 chiều, giúp căn chỉnh phần tử dễ dàng theo chiều ngang hoặc dọc.

.flex-container {
    display: flex;
    justify-content: center;  /* Căn giữa theo chiều ngang */
    align-items: center;      /* Căn giữa theo chiều dọc */
    flex-direction: row;      /* Hướng: row, column */
    gap: 10px;                /* Khoảng cách giữa các item */
}

📝 BÀI TẬP THỰC HÀNH

Câu 1: Làm thế nào để kích hoạt Flexbox trên một container?

display: flex;

Câu 2: justify-content dùng để làm gì? Kể tên các giá trị thường dùng.

Căn chỉnh các item theo trục chính. Các giá trị: flex-start, center, flex-end, space-between, space-around.

Câu 3: Làm sao để các item trong flex container hiển thị theo cột dọc?

flex-direction: column;

Câu 4: Thuộc tính nào tạo khoảng cách giữa các flex item? Có thể dùng thay thế bằng cách nào?

gap. Có thể dùng margin cho các item con.

Câu 5: align-items: center; có tác dụng gì? Khi nào nó hoạt động?

Căn giữa các item theo trục chéo (vuông góc với flex-direction). Hoạt động khi container có chiều cao xác định.

🔲 CSS Grid - Bố cục 2 chiều

📖 LÝ THUYẾT

Grid là hệ thống bố cục 2 chiều (hàng và cột), cho phép tạo layout phức tạp một cách dễ dàng.

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);  /* 3 cột đều nhau */
    grid-template-rows: auto;           /* Chiều cao tự động */
    gap: 20px;                      /* Khoảng cách giữa các ô */
}

📝 BÀI TẬP THỰC HÀNH

Câu 1: Làm thế nào để tạo grid với 4 cột có độ rộng bằng nhau?

grid-template-columns: repeat(4, 1fr);

Câu 2: 1fr trong Grid có ý nghĩa gì?

1fr là 1 phần không gian còn lại (fraction unit), giúp chia đều không gian.

Câu 3: Làm sao để tạo khoảng cách giữa các grid item?

gap: 20px; hoặc row-gap và column-gap riêng biệt.

Câu 4: Grid khác Flexbox như thế nào? Khi nào nên dùng Grid?

Grid làm việc trên 2 chiều (hàng và cột), Flexbox làm việc trên 1 chiều. Dùng Grid khi cần layout phức tạp.

Câu 5: Làm thế nào để một grid item chiếm 2 cột?

grid-column: span 2; hoặc grid-column: 1 / 3;

📱 Responsive Design

📖 LÝ THUYẾT

Responsive Design là thiết kế web thích ứng với mọi kích thước màn hình sử dụng Media Queries.

/* Desktop mặc định */
.container {
    display: flex;
    gap: 20px;
}

/* Tablet */
@media (max-width: 768px) {
    .container {
        flex-direction: column;
    }
}

/* Mobile */
@media (max-width: 480px) {
    .container {
        gap: 10px;
    }
}

📝 BÀI TẬP THỰC HÀNH

Câu 1: Media Query dùng để làm gì? Viết cú pháp cơ bản.

Áp dụng CSS khác nhau dựa trên kích thước màn hình. @media (max-width: 768px) { ... }

Câu 2: Viết media query cho màn hình có độ rộng từ 769px đến 1024px.

@media (min-width: 769px) and (max-width: 1024px) { ... }

Câu 3: Breakpoint phổ biến cho điện thoại và tablet là bao nhiêu?

Điện thoại: max-width 480px hoặc 768px, Tablet: 768px - 1024px.

Câu 4: Làm sao để ẩn một phần tử trên mobile?

@media (max-width: 768px) { .element { display: none; } }

Câu 5: viewport meta tag có tác dụng gì? Viết cú pháp đầy đủ.

<meta name="viewport" content="width=device-width, initial-scale=1.0"> - Giúp trang web hiển thị đúng trên thiết bị di động.

📝 BÀI TẬP THỰC HÀNH

Câu 1: transition dùng để làm gì? Cú pháp cơ bản?

Tạo hiệu ứng chuyển động mượt mà. transition: property duration timing-function;

Câu 2: Làm thế nào để tạo hiệu ứng khi di chuột vào phần tử?

Sử dụng pseudo-class :hover, ví dụ: .element:hover { transform: scale(1.1); }

Câu 3: transform: scale(1.2); có tác dụng gì?

Phóng to phần tử lên 1.2 lần kích thước ban đầu.

Câu 4: Làm sao để tạo hiệu ứng mờ dần khi hover?

transition: opacity 0.3s; .element:hover { opacity: 0.5; }

Câu 5: transition: all 0.3s ease-in-out; có ý nghĩa gì?

Tất cả thuộc tính thay đổi sẽ có hiệu ứng trong 0.3 giây với tốc độ chậm dần ở đầu và cuối.
