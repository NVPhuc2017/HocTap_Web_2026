JavaScript là gì? Ngôn ngữ lập trình kịch bản, giúp trang web trở nên tương tác và động.

📌 Vai trò của JavaScript:

  • ✓ Tạo hiệu ứng và tương tác động
  • ✓ Xử lý sự kiện từ người dùng (click, nhập liệu...)
  • ✓ Giao tiếp với máy chủ (AJAX, Fetch API)
  • ✓ Thay đổi nội dung và cấu trúc HTML/CSS
// Khai báo biến và in ra console

console.log(message);

📝 BÀI TẬP THỰC HÀNH

Câu 1: JavaScript dùng để làm gì trong phát triển web?

JavaScript tạo tương tác động, xử lý sự kiện, thay đổi nội dung HTML/CSS, và giao tiếp với máy chủ.

Câu 2: Có mấy cách để nhúng JavaScript vào HTML?

3 cách: Inline (trong thẻ), Internal (<script> trong HTML), External (file .js riêng).

Câu 3: console.log() dùng để làm gì?

In thông tin ra console của trình duyệt, dùng để debug và kiểm tra.

Câu 4: alert() có tác dụng gì?

Hiển thị hộp thoại thông báo cho người dùng.

Câu 5: JavaScript có phân biệt chữ hoa chữ thường không?

Có, JavaScript là ngôn ngữ case-sensitive (phân biệt hoa/thường).

Câu 1: Sự khác nhau giữa let, const và var?

let: khai báo biến có thể thay đổi, phạm vi block. const: hằng số không thể gán lại. var: phạm vi hàm, không khuyến khích dùng.

Câu 2: Kiểu dữ liệu nào dùng để lưu trữ danh sách các giá trị?

Array (mảng) - ví dụ: let list = [1, 2, 3];

Câu 3: typeof null trả về kết quả gì? Tại sao?

typeof null trả về "object" - đây là lỗi lịch sử của JavaScript.

Câu 4: Làm thế nào để kiểm tra kiểu dữ liệu của một biến?

Sử dụng toán tử typeof: typeof variableName

Câu 5: Khai báo một hằng số và một biến có thể thay đổi.

const PI = 3.14; let count = 0;

⚙️ Toán tử và Câu lệnh điều kiện

📖 LÝ THUYẾT
// Toán tử số học
let sum = 10 + 5; // 15
let product = 10 * 5; // 50

// Toán tử so sánh
let isEqual = (5 == "5"); // true (so sánh giá trị)
let isStrict = (5 === "5"); // false (so sánh cả kiểu)

// Câu lệnh if-else
let age = 18;
if (age >= 18) {
    console.log("Đủ tuổi");
} else {
    console.log("Chưa đủ tuổi");
}

// Switch case
let day = 2;
switch(day) {
    case 2: console.log("Thứ 2"); break;
    default: console.log("Ngày khác");
}

Câu 1: Sự khác nhau giữa == và === trong JavaScript?

== so sánh giá trị sau khi ép kiểu, === so sánh cả giá trị và kiểu dữ liệu.

Câu 2: Viết câu lệnh if-else để kiểm tra số dương, số âm, số 0.

if (num > 0) { console.log("Số dương"); } else if (num < 0) { console.log("Số âm"); } else { console.log("Số 0"); }

Câu 3: Toán tử ?: (ternary) dùng để làm gì?

Toán tử điều kiện rút gọn: condition ? valueIfTrue : valueIfFalse

Câu 4: Khi nào nên dùng switch thay vì if-else?

Khi cần so sánh một biến với nhiều giá trị cụ thể, switch giúp code rõ ràng hơn.

Câu 5: Viết code kiểm tra một số có chia hết cho 3 và 5 không?

if (num % 3 === 0 && num % 5 === 0) { console.log("Chia hết cho 3 và 5"); }

Câu 1: Sự khác nhau giữa function declaration và function expression?

Function declaration được hoisting (có thể gọi trước khi khai báo), function expression thì không.

Câu 2: Arrow function khác function thường ở điểm nào?

Arrow function ngắn gọn, không có this riêng, không thể dùng làm constructor.

Câu 3: Viết vòng lặp for để in ra số từ 1 đến 10.

for (let i = 1; i <= 10; i++) { console.log(i); }

Câu 4: Khi nào nên dùng while thay vì for?

While dùng khi không biết trước số lần lặp, for dùng khi biết trước số lần lặp.

Câu 5: Viết hàm tính giai thừa của một số.

function factorial(n) { if (n <= 1) return 1; return n * factorial(n-1); }
